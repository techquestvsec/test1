Coding

Codezilla

Individual Java programming — develop complete programs from rough draft to final submission across two timed sessions.

Language: Java
Mode: Individual
Max Participants: 8 per category
Categories: Junior (9–10) & Senior (11–12)
Overview

Participants individually develop programs in Java. Problems are revealed on the day. The event is split into a planning session and a coding session.

Session Structure
Session 1 — 1 Hour
Draft Phase
Participants think through their approach and write a rough draft of the programs they plan to code.
Session 2 — 3 Hours
Coding Phase
Participants implement their programs on a computer and submit the final code to the event head.
Judging Criteria
  • Efficiency — optimal use of logic and resources.
  • Functionality — correctness and completeness.
  • Optimization — clean, well-structured code.
