Design

Weave The Web

Build a website from scratch using only HTML and CSS — no scripting allowed, topic and assets provided on the day.

Languages: HTML & CSS only
Scripting: Not allowed
Mode: Individual / Team
Submission: Compressed file
Overview

Participants design and code a website on a provided topic using only HTML and CSS. All content and images are provided on the day. The final submission must be a compressed file.

Rules
  • Only HTML and CSS may be used — no JavaScript or any other scripting.
  • All provided content and images must be used as-is or adapted within the rules.
  • Submit the entire project as a compressed (.zip) file.
  • All code must be written from scratch during the event.
Judging Criteria
  • Design Quality — visual appeal, layout, and use of CSS.
  • Code Quality — clean, valid, well-structured HTML and CSS.
  • Creativity — originality in presentation and structure.
