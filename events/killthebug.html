Kill The Bug

20 Java snippets riddled with syntax, runtime, and logic errors — 4 hours to hunt and squash every bug.

Language: Java
Mode: Individual
Questions: 20
Duration: 4 Hours
Categories: Junior (9–10) & Senior (11–12)
Overview

Participants receive 20 buggy Java code snippets and must identify and correct all errors within 4 hours. Answers are submitted on a written answer sheet.

Error Types
  • Syntax errors — incorrect Java grammar or structure.
  • Runtime errors — code that crashes during execution.
  • Logic errors — code that runs but produces wrong output.
Grading
  • Points awarded for each runtime error reduced.
  • Points awarded for each syntax / logic error corrected.
  • Points deducted if the final snippet still contains errors.
  • Full points awarded if the corrected code runs without errors.
